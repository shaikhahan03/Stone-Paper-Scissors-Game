# Stone Paper Scissors Game 🎮

This is a simple Python-based Stone Paper Scissors game where the user plays against the computer.

## Features

* Menu-driven system
* Random computer choice using `random` module
* Score tracking
* Modular file structure

## Project Structure

```bash
Stone_Paper_Scissors/
│
├── main.py
├── play_game.py
├── result.py
├── utils.py
```

## How to Run

```bash
python main.py
```

## Requirements

* Python 3.x
